Help & Support

Getting Started

Authentication
  • Click "Log in to post" to create an account or sign in
  • Complete your profile setup after first login
  • Only authenticated users can create events
  • Use Google sign-in for quick access
Creating Events
  • Click "Post an event" when logged in
  • Or click on any calendar date to create an event for that day
  • Fill in required fields: title, start time, and end time
  • Add optional details like location, description, and images
  • Events are immediately visible to the community
Managing Events
  • Click any event to view full details
  • Edit or delete your own events from the event details view
  • Update events if details change
  • Cancel events by editing the description or deleting them
Navigation
  • Use arrow buttons to navigate between months
  • "Today" button returns you to the current date
  • Search upcoming events using the search box
  • View upcoming events in the sidebar
